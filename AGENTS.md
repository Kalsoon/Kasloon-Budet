# Prototype Instructions

Run the local server yourself and open the preview in the browser available to this environment. Do not give the user server-start instructions when you can run it.

The visual source of truth is the user-provided Cashly dashboard screenshot from 2026-07-14. Match its airy off-white canvas, white rounded cards, compact icon rail, warm coral accent, crisp typography, and dense but calm financial data presentation.

Product intent: Kalsoon is a Swiss budgeting fintech that helps people assign every CHF, escape paycheck-to-paycheck living, accelerate debt payoff, and build confidence with money.

Design preference: avoid dark filled accent surfaces. Use white or soft coral-tinted cards for emphasis, coral for selected navigation and controls, and reserve near-black for readable text only.
